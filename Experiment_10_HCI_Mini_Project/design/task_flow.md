# Task Flow Diagram

1. **Dashboard**
   - View summary → Go to Task Board / Analytics / Settings
2. **Task Board**
   - Add/Edit/Delete/Move tasks → Return to Dashboard
3. **Add Task Modal**
   - Save/Cancel → Return to Task Board
4. **Study Session Timer**
   - Start/Stop/Reset → Return to Dashboard
5. **Analytics**
   - View stats → Return to Dashboard
6. **Settings**
   - Change preferences → Return to Dashboard

**Navigation:**
- Main navigation bar allows switching between Dashboard, Tasks, Analytics, and Settings at any time.
- Modal overlays do not disrupt main navigation.
