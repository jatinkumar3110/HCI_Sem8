# UI Components

- **Task Card:** Displays task title, due date, priority, status, and actions (edit, delete, complete).
- **Kanban Column:** Holds task cards for each status (To Do, In Progress, Done).
- **Add Task Modal:** Form for entering new task details.
- **Timer Widget:** Pomodoro timer with controls.
- **Progress Bar:** Visualizes task completion.
- **Navigation Bar:** Tabs for Dashboard, Tasks, Analytics, Settings.
- **Notification Toast:** Shows feedback for user actions.
