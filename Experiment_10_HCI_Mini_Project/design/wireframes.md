# Wireframes

## 1. Dashboard
- Overview of tasks, productivity stats, and quick links.

## 2. Task Board
- Kanban board with columns: To Do, In Progress, Done.
- Each task is a card with title, due date, and status.

## 3. Add Task Modal
- Popup form to enter task title, description, due date, and priority.

## 4. Study Session Timer
- Pomodoro-style timer with start, pause, and reset buttons.

## 5. Analytics / Progress View
- Progress bars for completed tasks, pie chart for task distribution.

## 6. Settings
- Options for theme, notification preferences, and data reset.
