# Usability Testing Observations

**Participants:** 3 students

## Observations
- All users could add and edit tasks easily.
- Drag and drop was intuitive for most, but one user missed the visual cue.
- Timer was started successfully, but one user wanted a sound notification.
- Analytics were clear, but users wanted more detailed stats.

## Usability Issues
- Some buttons were small on mobile.
- Task due date picker could be more prominent.
- No confirmation before deleting a task.

## User Feedback
- "Love the clean design!"
- "Would like recurring tasks."
- "Progress bar is motivating."
