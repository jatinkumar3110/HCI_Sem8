# User Stories

1. As a student, I want to add study tasks so that I can track assignments.
2. As a student, I want to see upcoming deadlines so I do not miss submissions.
3. As a student, I want to organize tasks visually so I can manage my workload easily.
4. As a student, I want to mark tasks as completed so I can track my progress.
5. As a student, I want to use a timer to focus on study sessions.
6. As a student, I want to view my productivity stats so I can improve my study habits.
7. As a student, I want to edit and delete tasks so I can keep my planner up to date.
