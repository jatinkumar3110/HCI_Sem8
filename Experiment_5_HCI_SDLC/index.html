📋 User Requirements
Usability: Users should easily create, view, and delete notes without training
Accessibility: Clear labels, readable fonts, high contrast colors
Efficiency: Complete a note task in less than 30 seconds
Consistency: Same buttons, colors, and interactions throughout the app
🎯 HCI Principles Applied
Visibility: Status of notes clearly shown
Feedback: Confirmation messages for actions
Simplicity: Minimal, intuitive interface design
Control: Users can undo/delete any action
🎨 UI Design Principles
Layout: Vertical stack with clear sections (List, Input, Actions)
Color Scheme: Blue (#667eea) for primary actions, Red (#e74c3c) for delete
Typography: Bold headers, regular text for content, smaller text for hints
Spacing: 15px margins between elements for visual hierarchy
⚙️ Interaction Design
Buttons: Clear action labels (Add, Delete, Save)
Forms: Simple inputs with placeholders and labels
Feedback: Alerts confirm successful actions
Error Handling: Validation before adding notes
